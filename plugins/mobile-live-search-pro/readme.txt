Mobile-first popup AJAX live search with pagination, screenshot-style result cards, shortcode, and Elementor widget support.

== Usage ==
Shortcode: [mlsp_search]

Shortcode options:
[mlsp_search label="Search" placeholder="Search news..." size="32px" color="#111111" min_chars="2" per_page="8" post_types="post"]

Elementor:
Add the "Mobile Live Search Pro" widget, then customize the icon, popup, input, thumbnails, and result card styles.

== Notes ==
For best update results, delete any old mobile-live-search-pro folder before installing this ZIP.
